General Instructions for Counting Lights
  • Count any and all lights in home
  • A light is defined as every light bulb turned on by a single light switch.
  • A fixture or lamp with two switches is counted as two lights

Examples:

If a light has more than one bulb and more than one switch, count as two lights. Example: Floor Lamp.

If a light has only one switch, count as one light, even if it has more than one bulb. Example: Chandelier

If light can be turned on by a single switch, count it as one light. Example: track lighting.

State Energy Data System (SEDS): 2015 (updates by energy source)

New releases of energy consumption, price, and expenditure estimates by energy source. See Additional information.

A set of updated data files is presented under Data files. The "2015P" in the data_status field indicates that these are "in progress" estimates generated from the 2015 data cycle. These files incorporate the latest estimates and revisions that have been processed in SEDS. They are updated every time a new set of SEDS estimates is released.

Final data for the 2015 data cycle will be released in June 2017 with a data_status indicator of "2015F." See SEDS Complete for more information.
